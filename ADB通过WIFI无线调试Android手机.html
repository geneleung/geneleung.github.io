ADB通过WIFI无线调试Android手机

四 22 十一月 2018

主要以下两个命令，可以通过adb –help查看到 

adb tcpip PORT               restart adb server listening on TCP on PORT
# adb tcpip 5555
adb connect HOST[:PORT]      connect to a device via TCP/IP [default port=5555]
# adb connect x.x.x.x:5555

就这样很简单。
