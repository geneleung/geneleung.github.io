python3学习笔记

2018.12.10 星期一

tutorial

  • 代码块是用缩进控制的，缩进多少个space不重要，重要的是需要一致
  • 交互模式下>>> 与 …的区别，…需要用一个换行结束
  • 除法的返回值是浮点数，无论除数与被除数是什么类型，除法取整用//，取模用%
  • 幂运算符
  • 在交互模式下，最后一个表达式的值总会被赋给一个内建变量"_"，这个变量对用户只可读不可写。如果我们自己声明一个同名的变量，会覆盖内建变量。
  • 字符串
    • 对于''和""的使用和区别可以考虑lisp的对象，字符串是一个可被计算的值，他具备输入形式和输出形式，注意输出形式是对字符串值的表示形式，而不是打印字符串，打印字符串用print函数，输出形式和打印的区别在转义字符上体现的很明显，输出形式会输出\，而打印形式就会按具体转义字符的输出。

      输入形式'"的使用怎么方便怎么来就好。输出形式大部分情况使用'，只有以下一种情况下会使用"： 

      The string is enclosed in double quotes if the string contains a single quote and no double quotes, otherwise it is enclosed in single quotes.

      输出形式和打印形式看以下代码: 

        >>> 'It\'s\nhe said'
"It's\nhe said"
>>> print('It\'s\nhe said')
It's
he said
>>>

    python中的字符串还有一些小特点：

    • 不存在char类型，字符只是长度为1的字符串
    • 字符串是常量，不可被赋值，这通常出现在用下标索引时的赋值操作
    • 索引长度为1的字符串会越界，通过索引取字串不会出现越界
    • 字符串支持+ *的操作

函数

函数是什么

同lisp一样，函数也是有输出形式的，很类似，而且print也是个函数

变量作用域

也同lisp一样，似乎函数式语言的变量作用域总是很容易搞不明白，这里就搞的不是很明白，遇到问题可以再研究。尤其是参数默认值，要区分可变数据类型和不可变数据类型

函数的调用与参数列表

调用方式区别于其他语言，可以用key=value的形式调用函数，并指定参数。通过位置给定参数的形式一定要在通过key=value形式之前。

tuple(*name)参与与dict(**name)参数，tuple参数要在dict参数之前

数据类型

list

这种运算还是挺有意思的 

   vec = [-4, -2, 0, 2, 4]
# create a new list with the values doubled
[x*2 for x in vec]

# filter the list to exclude negative numbers
[x for x in vec if x >= 0]

# apply a function to all the elements
[abs(x) for x in vec]

# call a method on each element
freshfruit = ['  banana', '  loganberry ', 'passion fruit  ']
[weapon.strip() for weapon in freshfruit]

# create a list of 2-tuples like (number, square)
[(x, x**2) for x in range(6)]

# the tuple must be parenthesized, otherwise an error is raised
[x, x**2 for x in range(6)]




# flatten a list using a listcomp with two 'for'
vec = [[1,2,3], [4,5,6], [7,8,9]]
[num for elem in vec for num in elem]
