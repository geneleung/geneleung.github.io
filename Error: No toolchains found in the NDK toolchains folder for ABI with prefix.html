Error: No toolchains found in the NDK toolchains folder for ABI with prefix

四 22 十一月 2018

Android Studio升级到3.2.1之后在编译代码是遇到如下错误 

No toolchains found in the NDK toolchains folder for ABI with prefix mipsel-linux-android

原因是Android SDK在新版本中移除NDK对mipsel平台的支持，但是旧版的 android gradle plugin还在支持，在3.1版本及以上也已经移除了。通过修改 gradle脚本可以解决，如下： 

classpath "com.android.tools.build:gradle:3.2.0"

如果不想修改android gradle plugin版本可以通过如下方式解决： 

# on Mac
cd  ~/Library/Android/sdk/ndk-bundle/toolchains
ln -s aarch64-linux-android-4.9 mips64el-linux-android
ln -s arm-linux-androideabi-4.9 mipsel-linux-android
