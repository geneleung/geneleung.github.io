取消Mac OS X的磁盘加密

2020.04.20 星期一

用了一块1T的移动硬盘做为Macbook的Time Machine(时间机器)做备份，考虑到安全性勾选了“加密备份”，没想到备份时间虽不长，磁盘加密时间很长，需要以天为单位来计算了，而且是不仅仅是对备份内容加密，而是对整个磁盘加密，很不友好的是，一旦开始了这个过程用户界面上是无法取消这个过程的，只能等待加密结束。网上找到一个方法可以停止加密并解密，试了一下确实可用。

Yes, in 10.9+ you can roll it back while its busy encrypting 

diskutil cs revert /Volumes/title_drive -passphrase

You will then be promted with the OS X password unlock/decrypt dialog. After that it starts decrypting. The titledrive you can look up with diskutil list or ls -al /Volumes

Check the status with 

diskutil cs list

Look for "Conversion Progress" and check if "Conversion Direction:" is set to "backward".如果Status是Pause，重启一下电脑即可。另外，取消加密也是需要耗时的，这取决于已经加密了多少内容。
